A collection of basic C++ programs written as practice exercises.

Programs
• odd_numbers.cpp — Prints all odd numbers from 1 to n
• vowel_counter.cpp — Counts the number of vowels in a text input
• factorial.cpp — Calculates the factorial of a given number

Topics Covered
• Loops (for, while)
• Conditionals (if)
• User input/output (cin, cout)
• Data types (int, char, long long)
